## Twitter App
The Twitter app created should default to Casetab's latest 20 tweets
Searching for a user in the search bar up top will return the latest 20 tweets from said user and if the user mentions anyone, it will display the mentioned user. You can click on the user to view their twitter profile and latest 20 tweets as well.
